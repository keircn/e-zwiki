title: Image Border Settings
description: Add decorative borders and frames to your uploaded images
---

Image Border Settings let you add decorative borders and frames around your uploaded images.

## Border Type

Select from the following border styles:

| Type         | Description                                            |
| ------------ | ------------------------------------------------------ |
| **Solid**    | A simple solid color border                            |
| **Gradient** | A border with a color gradient effect                  |
| **Skeet**    | A rainbow gradient bar along the top edge of the image |
| **Glow**     | A glowing border effect around the image               |
| **Window**   | A window-style frame around the image                  |
| **Fade**     | A fading border effect                                 |
| **Mac**      | macOS-style window frame with traffic light buttons    |
| **Shadow**   | A drop shadow effect around the image                  |
| **Twitter**  | Twitter/X post-style frame with username and text      |

## Border Options

### Text

Custom text to display with certain border types (such as Twitter).

### Display Name

The name shown in the border frame for applicable border types.

### Verification Badge

Toggle to show a verification badge next to your display name (for supported border types like Twitter).

### Border Icon

Upload a custom icon or avatar image to display in the border frame.

## Enable Borders

Toggle to enable or disable image borders entirely. When disabled, images will be uploaded without any border effects.

## Saving Changes

Click **Save Changes** to apply your image border configuration.
