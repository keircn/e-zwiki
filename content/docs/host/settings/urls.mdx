URL Settings let you customize how your upload URLs are structured and displayed.

## URL Structure

Your upload URLs follow this format:

```
https://[subdomain].[domain]/[path]
```

| Component     | Description                                                    |
| ------------- | -------------------------------------------------------------- |
| **Subdomain** | Optional prefix before your domain (e.g., `i` in `i.e-z.host`) |
| **Domain**    | Your selected domain (e.g., `typescript.love`)                 |
| **Path**      | Optional custom path after the domain                          |

## URL Options

| Option              | Description                                                     |
| ------------------- | --------------------------------------------------------------- |
| **Long URL**        | Use a longer, more unique filename in the URL                   |
| **Fake URL**        | Display a fake URL in embeds (uses the `{fakeurl}` placeholder) |
| **Show Link**       | Show the direct link in the embed                               |
| **Emoji Filename**  | Use emojis as the filename in the URL                           |
| **Custom Filename** | Set a custom filename for your uploads                          |
| **Sus Filename**    | Use a suspicious-looking filename (for humorous effect)         |
| **Invis Filename**  | Use invisible/zero-width characters as the filename             |
| **Random Domain**   | Randomly select from your available domains for each upload     |
| **Custom Webhook**  | Send upload notifications to a custom Discord webhook           |

## Saving Changes

Click **Save Changes** to apply your URL settings configuration.
