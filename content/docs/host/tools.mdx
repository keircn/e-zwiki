---
title: Upload Tools
description: Download upload tools for Windows, MacOS, iOS, Linux, and Android
---

E-Z.host provides native upload tools for all major platforms. Click on your platform in the dashboard to download the appropriate tool or configuration.

## Windows

**Tool:** [ShareX](https://getsharex.com/)

ShareX is a free and open-source screenshot and file sharing tool for Windows. Click the **Download ShareX Config** button to download a pre-configured `.sxcu` file that sets up E-Z.host as your upload destination.

1. Download and install ShareX
2. Download the ShareX config from your E-Z dashboard
3. Double-click the `.sxcu` file to import it into ShareX
4. ShareX will now upload directly to E-Z.host

## MacOS

**Tool:** E-Z Desktop App (Tauri)

A native desktop application built with Tauri for macOS users. Download the app directly from the dashboard to get started.

## iOS

**Tool:** iOS Shortcut

Use the iOS Shortcuts app to upload files and screenshots directly to E-Z.host. Download the shortcut from the dashboard and add it to your Shortcuts app.

## Linux

**Tool:** Flameshot + Custom Script

Linux users can use [Flameshot](https://flameshot.org/) for screenshots combined with a custom upload script.

1. Install Flameshot using your package manager
2. Download the upload script from your E-Z dashboard
3. Configure your keybinds to run the script after capturing
